# Input_MonkeyWrench
Simple script(s) for using Arduino Micro as keyboard HID for a number of input projects.

Each pin is `INPUT_PULLUP` — short pin to GND to trigger.

## Pin Map

| Pin | Key          | Typical Use            |
|-----|--------------|------------------------|
| 2   | Space        | Morse key / jump       |
| 3   | Enter        | Confirm / send         |
| 4   | Esc          | Menu / cancel          |
| 5   | Left Ctrl    | Crouch / modifier      |
| 6   | Left Shift   | Sprint / modifier      |
| 7   | Tab          | Map / scoreboard       |
| 8   | W            | Forward                |
| 9   | A            | Strafe left            |
| 10  | S            | Back                   |
| 16  | D            | Strafe right           |
| 14  | E            | Interact / use         |
| 15  | R            | Reload                 |
| 18  | ↑ Arrow      | Up                     |
| 19  | ↓ Arrow      | Down                   |
| 20  | ← Arrow      | Left                   |
| 21  | → Arrow      | Right                  |
