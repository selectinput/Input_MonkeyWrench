# Input_MonkeyWrench
Simple script(s) for using Arduino Micro as keyboard HID for a number of input projects.

Each pin is `INPUT_PULLUP` — short pin to GND to trigger.

## Pin Map

| Pin | Key          |
|-----|--------------|
| 2   | Space        |
| 3   | Enter        |
| 4   | Esc          |
| 5   | Left Ctrl    |
| 6   | Left Shift   |
| 7   | Tab          |
| 8   | W            |
| 9   | A            |
| 10  | S            |
| 16  | D            |
| 14  | E            |
| 15  | R            |
| 18  | ↑ Arrow      |
| 19  | ↓ Arrow      |
| 20  | ← Arrow      |
| 21  | → Arrow      |
